# RecycleFashion ♻️

**Earn vouchers for new clothes by recycling old ones.**  
Take a photo, enter the weight, get 10 points per kg → 100 points = **£10 voucher**.

## Features (MVP)
- Photo upload (camera / gallery)
- Weight input
- Auto-approval (demo)
- Persistent points & voucher list (SharedPreferences)
- Copy-to-clipboard vouchers
